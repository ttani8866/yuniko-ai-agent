# 日本語フォント設定ガイド

## video_generator.pyの場所

`video_generator.py`は以下の場所にあります：
```
01_Applications/04_tubeauto/video_generator.py
```

## Windowsでフォントファイルを探す手順

### 方法1: エクスプローラーのアドレスバーで開く（最も簡単）

1. **エクスプローラーを開く**
   - Windowsキー + E を押す
   - または、タスクバーのフォルダアイコンをクリック

2. **アドレスバーをクリック**
   - エクスプローラーの上部にある、現在のフォルダパスが表示されている部分をクリック

3. **以下のパスを入力してEnterキーを押す**:
   ```
   C:\Windows\Fonts
   ```
   または
   ```
   C:/Windows/Fonts
   ```

4. **フォントファイルの一覧が表示されます**

### 方法2: ファイル名を指定して実行（Win+R）

1. **Windowsキー + R を押す**
   - 「ファイル名を指定して実行」ダイアログが開きます

2. **以下のパスを入力してEnterキーを押す**:
   ```
   C:\Windows\Fonts
   ```

3. **フォントフォルダが開きます**

### 方法3: コマンドプロンプトから開く

1. **コマンドプロンプトを開く**
   - Windowsキー + R を押す
   - `cmd` と入力してEnterキーを押す

2. **以下のコマンドを実行**:
   ```cmd
   explorer C:\Windows\Fonts
   ```

3. **フォントフォルダが開きます**

### 方法4: 検索から開く

1. **Windowsキーを押してスタートメニューを開く**

2. **検索ボックスに「フォント」と入力**

3. **「フォント設定」または「Fonts」をクリック**

4. **フォントフォルダが開きます**

## よく使われる日本語フォント

フォントフォルダが開いたら、以下のファイルを探してください：

- **`msgothic.ttc`** - MS ゴシック（推奨）
- **`msmincho.ttc`** - MS 明朝
- **`meiryo.ttc`** - メイリオ
- **`yugothic.ttf`** - 游ゴシック

**注意**: ファイル名の拡張子（`.ttc`や`.ttf`）も含めて確認してください。

## video_generator.pyの設定方法

1. **`01_Applications/04_tubeauto/video_generator.py`を開く**

2. **14行目あたりの`FONT_PATH = None`を探す**

3. **見つけたフォントのパスを設定**。例：
   ```python
   FONT_PATH = "C:/Windows/Fonts/msgothic.ttc"  # MS ゴシック
   ```
   または
   ```python
   FONT_PATH = "C:/Windows/Fonts/meiryo.ttc"  # メイリオ
   ```

   **重要**: パスはスラッシュ（`/`）で区切ってください。バックスラッシュ（`\`）の場合は、`r"C:\Windows\Fonts\msgothic.ttc"`のように`r`を前に付けるか、`C:/Windows/Fonts/msgothic.ttc`のようにスラッシュに変更してください。

4. **ファイルを保存**

## 自動検出について

`video_generator.py`は、よく使われるフォントのパスを自動的に検出しようとします。以下のフォントが自動検出されます：

- `C:/Windows/Fonts/msgothic.ttc` - MS ゴシック
- `C:/Windows/Fonts/msmincho.ttc` - MS 明朝
- `C:/Windows/Fonts/meiryo.ttc` - メイリオ
- `C:/Windows/Fonts/yugothic.ttf` - 游ゴシック

これらのフォントが存在する場合、自動的に使用されます。存在しない場合は、上記の手順で手動設定してください。

## 確認方法

設定後、Streamlitアプリを起動すると、サイドバーにフォントの検出状況が表示されます：
- ✅ フォント検出: [フォント名] → 正常に検出されています
- ⚠️ 日本語フォントが見つかりません → 手動設定が必要です

## トラブルシューティング

### フォントファイルが見つからない場合

1. **ファイル名の確認**
   - 拡張子（`.ttc`や`.ttf`）も含めて正確に入力してください
   - 大文字・小文字は区別されませんが、スペルを確認してください

2. **パスの確認**
   - `C:\Windows\Fonts` が正しいパスか確認してください
   - スラッシュ（`/`）とバックスラッシュ（`\`）の違いに注意してください

3. **ファイルの存在確認**
   - コマンドプロンプトで以下のコマンドを実行して確認：
     ```cmd
     dir C:\Windows\Fonts\msgothic.ttc
     ```
