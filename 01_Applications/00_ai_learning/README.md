# バイブコーディング学習管理アプリ

## 概要
バイブコーディングを段階的に習得するための「自分専用Web管理アプリ」

### 🔧 機能一覧

| 機能 | 説明 |
|------|------|
| レベル別表示 | 学習項目をレベル（入門/基礎/応用/実践）ごとに分類表示 |
| チェックボックス | 各項目の達成状況をON/OFF切り替え |
| 進捗バー | レベル別・全体の進捗率を視覚的に表示 |
| 状態保存 | localStorageでチェック状態を自動保存 |
| 折りたたみ | レベルごとに展開/折りたたみ可能 |

### 🛠 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| 言語 | HTML + CSS + JavaScript | 初心者でも理解しやすい |
| フレームワーク | なし（Vanilla JS） | 依存なしでシンプル |
| スタイリング | CSS変数 + Flexbox | モダンかつ軽量 |
| データ保存 | localStorage | サーバー不要で永続化 |
| 開発環境 | Cursor / Antigravity / Replit / bolt / v0 対応 | 1ファイルでも動作可能 |

## 使い方

1. `index.html` をブラウザで開く
2. 学習項目をクリックしてチェック
3. 進捗が自動保存される
4. ブラウザを閉じても状態は保持

## 開発環境での実行

### Cursor（推奨）
1. フォルダを開く
2. `index.html` を右クリック →「Open with Live Server」または直接ブラウザで開く
3. AIチャットでコードの修正・拡張を依頼

### Antigravity
1. 新規プロジェクト作成
2. ファイルをアップロード
3. プレビューで確認

### Replit
1. 新規HTMLプロジェクト作成
2. ファイルをコピー
3. Run ボタンで実行

### bolt / v0
1. 新規プロジェクト作成
2. コードをペースト
3. プレビューで確認

### ローカル
1. フォルダをダウンロード
2. `index.html` をブラウザで開く（ダブルクリック）
