# バイブコーディング短期講座 LP

AIを用いた開発であるバイブコーディングの短期講座のランディングページです。

## プロジェクト構成

```
05_LP/
├── index.html              # メインHTMLファイル
├── styles.css              # スタイルシート
├── script.js               # JavaScriptファイル
├── README.md               # このファイル
├── 01_競合調査結果.md      # 競合調査の結果
├── 02_LPセクション構成.md  # LPのセクション構成
└── 03_LPデザイン設計.md    # デザイン設計書
```

## セクション構成

1. **ヘッダー**: ナビゲーションとCTAボタン
2. **ヒーローセクション**: メインキャッチコピーと統計情報
3. **問題提起セクション**: ターゲットの課題を提示
4. **解決策セクション**: バイブコーディングの説明
5. **講座の特徴セクション**: 5つの特徴をカード形式で表示
6. **カリキュラムセクション**: 4週間の学習内容をタイムライン形式で表示
7. **受講生の声セクション**: テスティモニアルスライダー
8. **講師紹介セクション**: 講師のプロフィール
9. **料金プランセクション**: 3つのプランを比較表示
10. **FAQセクション**: よくある質問をアコーディオン形式で表示
11. **最終CTAセクション**: 行動喚起
12. **フッター**: リンクと連絡先情報

## 機能

- ✅ レスポンシブデザイン（モバイル・タブレット・デスクトップ対応）
- ✅ スムーススクロール
- ✅ モバイルメニュー（ハンバーガーメニュー）
- ✅ 受講生の声スライダー（自動スライド）
- ✅ FAQアコーディオン
- ✅ スクロールアニメーション（フェードイン）
- ✅ 固定ヘッダー（スクロール時に影を追加）

## 使用方法

1. `index.html`をブラウザで開く
2. ローカルサーバーで実行する場合:
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsの場合
   npx serve
   ```

## カスタマイズ

### カラーパレットの変更

`styles.css`の`:root`セクションでカラーを変更できます:

```css
:root {
    --color-primary: #6366F1;  /* メインカラー */
    --color-accent: #F59E0B;   /* アクセントカラー */
    /* ... */
}
```

### コンテンツの変更

`index.html`の各セクションのコンテンツを編集してください。

### 画像の変更

プレースホルダー画像（`https://placehold.co/...`）を実際の画像URLに置き換えてください。
